
## 📌 팀 프로젝트 caffeshop 웹사이트


![caffeshop](https://github.com/user-attachments/assets/913e3075-6ab6-4bd4-87f2-713d44587e0d)


▷ 팀 프로젝트로 진행한 카페 브랜드  포트폴리오 웹사이트입니다.  

▷ 국내에서 오랜 역사를 가진 카페 브랜드를 선정하여 기존 웹사이트를 분석하고 리뉴얼 프로젝트를 진행했습니다.

▷ 실제 사용자 경험이 다소 부족해 보이며, 반응형 웹이 적용되지 않아 다양한 디바이스에서의 접근성이 떨어지는 문제가 있어, 이를 개선하기 위해 UI 간격과 가독성을 재정비 하여 반응형 웹으로 리뉴얼 하였습니다.



<br>
