
## 📌 팀 프로젝트 caffeshop 웹사이트


![caffeshop](https://github.com/user-attachments/assets/913e3075-6ab6-4bd4-87f2-713d44587e0d)


▷ 팀 프로젝트 포트폴리오 웹사이트입니다.  

▷ 국내 브랜드중 제일 오래 된 카페

▷ 실제 사용자 경험이 부족해 보이며, UI 요소 간격 및 가독성이 떨어져 리뉴얼을 통한 반응형으로 제작 하게 되었습니다.

## 📌 프로젝트 주요 기능

- Fullpage 스크롤 기반 페이지 구성
- GSAP ScrollTrigger 애니메이션
- 프로젝트 카드 Hover 인터랙션
- 반응형 레이아웃

project

 │
 ├ css
 │   └ main.css 
 │   └ mainPDP.css 
 │   └ mainSub.css 
 ├ js
 │   └ main.js
 ├ home
 │   └ img
